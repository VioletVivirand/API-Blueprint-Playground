功能修改備忘錄 ¶

在本案中，POST 的部分應該全部改為 GET。

雖然可以混著用，不過在一般 API 上面看到的慣例，POST 方法的請求通常會拿來修改資料，而回傳的 HTTP Status Code 則會是 201。

雖然用 POST 方法有可以把傳送的值給包裝起來的用途，不過在這個案子裡並沒有太多的意義。
